# ICar Graphical User Interface

Menggunakan RVIZ sebagai sebuah widget dalam GUI dengan platform Qt. RViz sendiri adalah visualizer 3D yang digunakan dalam ROS (Robot Operating System) untuk memvisualisasikan data sensor dan status robot. Berikut beberapa fitur yang disuguhkan oleh aplikasi GUI Icar dengan pemanfaatan ROS dan Qt:

- Visualisasi Model Mobil
- Visualisasi Speedomter
- Visualisasi Sebuuah Jam realtime
